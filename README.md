# Bus Duty

A 3D bus driving simulation game built in Unity where the player picks up students and completes routes.

## About the Game
Bus Duty is a 3D bus driving mobile game. The player drives a bus through routes to pick up students and complete the route successfully.

## Gameplay
- Drive the **bus** through the route
- **Pick up students** at their stops
- Follow **waypoints** to reach each stop
- Complete the **route** successfully

## Features
- Bus driving controls
- Waypoint navigation system
- Student pickup system
- Mobile-ready controls

## Built With
- **Engine:** Unity
- **Language:** C#
- **Platform:** Mobile (Android)
